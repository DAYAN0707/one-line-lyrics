# One Line Lyrics

自分の中の「推し歌詞」をコレクション・共有するWebアプリです。

## 実装機能
* **ユーザー認証:** 登録・ログイン・ログアウト機能
* **歌詞管理:** 投稿の閲覧、新規作成、**編集機能**
* **マイページ機能:** 自分の投稿にのみ編集ボタン表示
* **こだわりのUI:** アーティストのライブ感をイメージしたネオンカラーとダークモードのCSSデザイン

## 使用技術
* **Backend:** Python / Django
* **Frontend:** HTML5 / CSS3
* **Tools:** Git / GitHub / VS Code

## デザインのこだわり
ライヴ会場を連想する世界観を表現するため、背景をブラック設定、アクセントカラーは鮮やかネオン（`#ccff00`）を採用
投稿カードに光彩（box-shadow）のエフェクトを加え、ライブの熱量を感じさせるデザインを意識
